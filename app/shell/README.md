# SolOS Shell

Initial shell scaffold for the SolOS MVP.

## Goal

Create the first believable operating-layer interface for SolOS.

## Initial scope

- App shell
- Home
- Agent
- Wallet
- Apps
- Shared state model
- Reusable UI components

## Suggested stack

- Tauri
- React
- TypeScript
- Zustand

## Status

Structure scaffold only. Implementation begins from the shell base and Home screen.
